# Fumo Game

## Lancer l'API
```bash
# Se placer dans le dossier de l'API
cd fumoAPI

# Installer les packages
npm install

# Installer nodemon en global
npm install --global nodemon

# Créer la base de données
node databases/migrations.js

# Démarre l'API en local sur le port 3000
npm start
```

## Lancer l'app
```bash
# Se placer dans le dossier de l'application
cd 'Fumo Game'

# Installer les packages
npm install

# Démarre l'application en local
npm run dev
```
